# NgramParser
Parse a file and return word frequencies, predict the patterns.

1. Create virtual env
    `python3 -m venv .venv`

2. Load virtual env
    `source .venv/bin/activate`

3. Install fastapi
    `pip install fastapi`

4. Install uvicorn
    `pip install "uvicorn[standard]"`

5. Run the app
    `uvicorn main:app --reload`

6. Install pytest
    `pip install -U pytest`

7. Run tests
    `pytest`
